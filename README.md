# pandas-data-analysis
# Pandas Practice

A collection of hands-on exercises and examples to learn and practice **Pandas**, one of the most powerful Python libraries for data manipulation and analysis.

## 📚 Topics Covered

- Introduction to Pandas
- Series and DataFrames
- Reading and Writing Data (CSV, Excel)
- Data Inspection
- Indexing and Selection
- Filtering Data
- Handling Missing Values
- Sorting and Ranking
- GroupBy Operations
- Merging and Joining DataFrames
- Data Cleaning
- Basic Data Analysis

## 🛠️ Technologies Used

- Python
- Pandas
- Google Colab / Jupyter Notebook

## 🎯 Objective

This repository is created to strengthen my understanding of Pandas through practical examples and exercises.

## 🚀 How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/pandas-practice.git
   ```

2. Open the notebook in:
   - Google Colab
   - Jupyter Notebook

3. Install Pandas if needed:
   ```bash
   pip install pandas
   ```
